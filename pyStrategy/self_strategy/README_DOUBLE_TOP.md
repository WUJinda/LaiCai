# 双峰做空策略 (Double Top Short Strategy)

## 策略概述

日线周期做空策略，基于双顶形态 + 左侧交易思路。当价格经历一个显著高点（左峰）后回调到布林中轨，再次反弹接近左峰区间时直接做空，价格回到中轨时止盈。

## 核心逻辑

```
布林带宽 > 10% → 价格回调到中轨 → 回溯30日找左峰(H_left)
→ 价格反弹到 [0.99×H_left, 1.02×H_left] → 左侧交易直接做空
→ 价格回到中轨 → 平仓止盈
```

### 前滤条件
- 布林带(20, 2.0) 带宽必须 > 中轨的 10%
- 避免在布林带极度收窄（squeeze）时产生信号

### 左峰识别
- 价格收盘 ≤ 布林中轨时触发
- 向前回溯 30 根日K线，取最高价作为左峰 H_left

### 入场（左侧交易）
- 价格进入 [0.99 × H_left, 1.02 × H_left] 区间
- 立即做空，不等确认信号
- 左侧交易的逻辑：在阻力位提前入场，胜率高但可能承受短期继续上涨的风险

### 止盈
- 价格触及布林中轨（20日均线）时全仓平仓
- 中轨代表阶段性的平均成本，价格回到成本区时获利了结

### 止损
- **不设止损**（需注意极端行情风险）

### 形态失效
- 在等待右峰区间时，价格突破 1.02 × H_left → 形态失效，回到监控初始状态

## 状态机

| 状态 | 含义 | 行为 |
|------|------|------|
| IDLE (0) | 空闲等待 | 等带宽达标 |
| WAITING_PULLBACK (1) | 等待回调 | 等价格触及中轨 |
| LEFT_PEAK_FOUND (2) | 左峰确认 | 等价格回到区间做空 / 突破失效 |
| IN_POSITION (3) | 持仓中 | 等价格回到中轨止盈 |

## 参数说明

### 布林带参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `bb_period` | 20 | 布林带周期 |
| `bb_std` | 2.0 | 标准差倍数 |
| `bandwidth_min` | 0.10 | 最小带宽（中轨的10%） |

### 左峰参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `left_peak_lookback` | 30 | 回溯K线数查找左峰 |

### 入场区间参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `zone_lower` | 0.99 | 区间下沿（左峰倍数） |
| `zone_upper` | 1.02 | 区间上沿（左峰倍数） |

### 交易参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| `pay_up` | 0 | 超价（≥0 提高成交率但增加滑点） |
| `margin_rate` | 0.10 | 保证金率（10%） |

## 资金管理

| 规则 | 数值 |
|------|------|
| 总保证金 | 1000 万 |
| 单笔保证金上限 | 100 万 |
| 同时持仓保证金上限 | 600 万 |
| 手数计算 | 动态计算（价格 × 合约乘数 × 保证金率） |

## 使用建议

1. **品种选择**: 选择流动性好的期货品种，避免小品种的剧烈波动
2. **参数调整**: `bandwidth_min` 控制信号频率，调高=更少信号但更可靠
3. **区间参数**: `zone_lower/zone_upper` 控制入场宽松度，默认 [0.99, 1.02] 较保守
4. **无止损注意**: 建议在品种选择上避开单边强势上涨的品种
5. **日线周期**: 信号较少，需要耐心等待，适合中长线交易
6. **先模拟测试**: 建议在模拟盘充分验证后再实盘运行

## 与现有 BOLL 策略的区别

| 维度 | BOLL 做空策略 | 双峰做空策略 |
|------|-------------|------------|
| 周期 | H2 (2小时) | D1 (日线) |
| 开仓逻辑 | 价格突破布林上轨 | 价格回到左峰区间 |
| 入场方式 | 右侧（突破确认） | 左侧（区间入场） |
| 核心形态 | 布林带扩张突破 | 双顶 + 中轨回调 |

## 技术指标说明

### 布林带（Bollinger Bands）
- 中轨: 20日简单移动平均线（SMA）
- 上轨: 中轨 + 2倍标准差
- 下轨: 中轨 - 2倍标准差

### 带宽
```
带宽 = (上轨 - 下轨) / 中轨
```
带宽反映了布林带的宽度相对于价格的水平，带宽越大说明波动率越高。

## 常见问题

Q: 为什么不做止损？
A: 策略核心假设是价格在双峰区间会受阻回落，中轨止盈。未来可根据回测结果决定是否加入止损。

Q: 如果价格一直不回到中轨怎么办？
A: 策略会一直持有空仓直到价格回到中轨。在强下跌趋势中这会带来较大利润，但也意味着浮盈可能回吐。

Q: 左峰30天的窗口可以调整吗？
A: 可以，`left_peak_lookback` 参数可调。窗口越大，找到的左峰越"显著"，但可能会漏掉较短期的双顶。

Q: 左侧交易的风险是什么？
A: 左侧交易是在阻力位直接入场，可能"抄在半山腰"——即价格继续上涨突破左峰。策略通过形态失效条件（突破 1.02×H_left）来处理这种情况。
